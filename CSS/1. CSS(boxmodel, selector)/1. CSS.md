## BoxModel
![image](https://github.com/SeokJuGo/SSAFY_TIL/assets/116260619/816f38ed-9343-4e48-b8df-c5cf0895fdbe)
content- 텍스트와 이미지가 표시되는 상자의 콘텐츠
padding - 콘텐츠 주변 영역을 지웁니다. 패딩은 투명합니다
border - 패딩과 콘텐츠를 둘러싸는 테두리
margin- 테두리 밖의 영역을 지웁니다. 여백이 투명하다

## Selector

### 1. Universal
```html
* {
  text-align: center;
  color: blue;
}
```
### 2. Element
```html
p {
  text-align: center;
  color: red;
}
```
### 3. Class
```html
.center {
  text-align: center;
  color: red;
}
```
```html
p.center {
  text-align: center;
  color: red;
}
```
### 4. ID
```html
#para1 {
  text-align: center;
  color: red;
}
```

### 5. 자식, 하위 선택자
